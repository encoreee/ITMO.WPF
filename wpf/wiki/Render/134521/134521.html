Southridge Hands-on-Lab

Southridge is an extensive hands-on-lab for WPF developers. The six-part lab includes the following sections:
  1. Fundamentals
  2. Custom Controls
  3. Styling
  4. Ribbon
  5. DataGrid
  6. Databinding

Download the Southridge lab on the WPF Futures page.

Please note that in order to use the Ribbon sections of this lab, you must install the RibbonControlsLibrary.dll from the Office UI Licensing Site using the instructions on this page. Due to licensing requirements, RibbonControlsLibrary.dll is not included with the Southridge lab's zip file and must be downloaded separately.
